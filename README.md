# pytorch-mini-lstm-handson
I once read through the LSTM code in this repository: https://github.com/nicodjimenez/lstm. It's really well written and helps you understand how LSTMs work from the ground up. However, it's a bit too low-level — even backpropagation has to be manually implemented with custom functions.

That's why I wrote a version that lightly leverages the PyTorch framework to handle the dirty work for me, while still allowing you to understand the underlying principles.
